# 🎓 Education25 – E-Learning Platform

A complete E-Learning Web Application where students can browse courses, purchase paid courses, 
attempt online tests, view results, and download certificates.

---

## 🚀 Project Overview

Education25 provides a full-featured digital learning system including:

- Course Management
- Online Examination Module
- Certificate Generation
- Advanced Calculator System
- Admin Dashboard

---

## 👨‍🎓 User Features

### 📚 Course Section

- Browse all available courses with complete details
- Purchase paid courses via integrated payment system
- Access free courses with modular learning content

---

### 🧮 Advanced Calculator Module

#### ➤ Basic Calculator

- Addition (+)
- Subtraction (–)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- History tracking

#### ➤ Matrix Operations

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Determinant
- Transpose
- Cofactor
- Adjoint
- Inverse

#### ➤ Other Mathematical Tools

- Factorial Calculator
- Circle Calculations
- Quadrilateral Calculations

---

### 📝 Online Examination System

- Attempt online exams
- Instant result generation
- Automatic score calculation

---

### 🏆 Certificate Generation

- Auto-generated certificates
- Downloadable after successful completion

---

## 🛠️ Admin Panel Features

### 📚 Course Management

- Add new courses
- Update course content
- Upload and update course images
- Delete / Deactivate courses

### 📝 Exam Management

- Create question papers
- Add, update, delete questions
- Add / update images in exams

## 👨‍💻 My Role (Personal Project)

Education25 is a fully self-developed personal project where I handled the complete development lifecycle from design to deployment.

- Designed and developed the complete frontend (HTML, CSS, JavaScript)
- Implemented backend logic using Java (JSP & Servlets)
- Designed relational database schema in MySQL
- Developed dynamic course and exam modules
- Built automated certificate generation system
- Integrated advanced mathematical calculator features
- Managed deployment on Apache Tomcat

---

## 💻 Tech Stack

- Backend: Java (JSP & Servlets)
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Server: Apache Tomcat
